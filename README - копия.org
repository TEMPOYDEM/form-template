1. Зарегистрироваться на [[https://getsimpleform.com/]]
2. Получить 2 api token \mdash уникальных ключа для работы с сервисом
3. Форкнуть этот репозиторий
4. =git clone ссылка=
5. Изменить api token (см. код)
6. Изменить редирект или удалить его
7. Изменить цвет
8. Изменить шрифт
9. Изменить дизайн в целом по Вашему усмотрению.
10. Когда закончите работу над страницей:
    1. git add *
    2. git commit -m 'ваше сообщение'
    3. git push
